# simple_test - Fullstack

## Highlight
- Use React hooks.
- Form validation with Formik.

## Getting Started

1. Install your dependencies

    ```
    yarn install
    ```
2. Build client

    ```
    yarn build-client
    ```
    
2. Run the application

    ```
    yarn start
    ```
